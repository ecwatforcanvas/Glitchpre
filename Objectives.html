>_ENGLISH COURSE WITH AI TOOLS_OBJECTIVES_

> GENERAL_OBJECTIVE

TO DEVELOP A BASIC ENGLISH II COURSE WITH AI TOOLS ON THE CANVAS PLATFORM, ENHANCING LANGUAGE ACQUISITION THROUGH INTERACTIVE, AI SUPPORTED TOOLS.

> SPECIFIC_OBJECTIVES

  • TO CREATE AN INSTRUCTIONAL DESIGN THAT PROVIDES ENGAGING AND INTERACTIVE ACTIVITIES TO SUPPORT STUDENTS' LEARNING.
  • TO DESIGN INTERACTIVE ACTIVITIES USING CHATGPT, DEEPSEEK, GEMINI, AND COPILOT, ENHANCING STUDENT ENGAGEMENT AND PROVIDING DYNAMIC LANGUAGE PRACTICE.
  • TO CREATE GRAMMAR INFOGRAPHICS WITH CANVA DESIGN EDITOR, FACILITATING VISUAL LEARNING AND REINFORCING KEY LANGUAGE CONCEPTS.
  • TO PRODUCE INSTRUCTIONAL VIDEOS USING CAPCUT, IMPROVING COMPREHENSION THROUGH MULTIMEDIA RESOURCES.
  • TO DEVELOP INTERACTIVE PRESENTATIONS ON CANVA, SUPPORTING STRUCTURED AND VISUALLY APPEALING LESSON DELIVERY.
  • TO DESIGN H5P-BASED ACTIVITIES USING LUMI, ENABLING INTERACTIVE EXERCISES FOR ENHANCED STUDENT PARTICIPATION.
  • TO GENERATE KEY VOCABULARY PDFS, AIDING IN VOCABULARY RETENTION AND QUICK REFERENCE FOR STUDENTS.
  • TO CRAFT GUIDED INSTRUCTIONS FOR VOICEMAKER, HELPING STUDENTS IMPROVE PRONUNCIATION AND LISTENING SKILLS THROUGH TEXT-TO-SPEECH TECHNOLOGY.
  • TO PROVIDE STEP-BY-STEP GUIDES FOR LEONARDO.AI, ALLOWING STUDENTS TO CREATE AI-GENERATED VISUALS FOR CREATIVE LANGUAGE PRACTICE.
  • TO DEVELOP AN HTML VIDEO GAME AS A FINAL PROJECT, REINFORCING COURSE CONTENT THROUGH GAMIFIED LEARNING.
  • TO STRUCTURE A LEARNING PATH ON CANVAS, ORGANIZING CONTENT FOR A SEAMLESS AND PROGRESSIVE LEARNING EXPERIENCE.
  • TO IMPLEMENT CONTROL QUIZZES FOR EACH MODULE, ENSURING COMPREHENSION AND RETENTION OF KEY CONCEPTS.
