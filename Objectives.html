ENGLISH COURSE WITH AI TOOLS // OBJECTIVES

> General_objective

To develop a basic English II course with AI tools on the Canvas platform, enhancing language acquisition through interactive, AI supported tools.

> Specific_objectives ▶

  • To create an instructional design that provides engaging and interactive activities to support students' learning.
  • To design engaging and interactive activities using tools such as ChatGPT, DeepSeek, Gemini, Copilot, Canva, H5P (via Lumi), and HTML games, promoting dynamic language practice and student participation.
  • To create grammar infographics with Canva, facilitating visual learning and reinforcing key language concepts.
  • To produce instructional videos using CapCut, enhancing comprehension through multimedia resources.
  • To provide step-by-step guides for Leonardo.AI, encouraging creative language practice through AI-generated visuals.
  • To implement control quizzes for each module, monitoring comprehension and reinforcement of key concepts.
